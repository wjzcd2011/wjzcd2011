CINEMA 4D 插件TRYPOGEN

发表于2021-04-20|更新于2022-09-20|插件
|字数总计:145|阅读时长:1分钟|阅读量:

Trypogen 1.5 for Cinema 4D是一款由Merkvilson出品的C4D多边形镂空插件，可以调节镂空厚度，平滑，尖，平滑细分等参数，简单快捷制作抽象镂空效果，使用细分编辑器Subdivision Editor可以预览效果，Subdivision Renderer可以查看最终效果.Trypogen plugin for Cinema 4D? Maybe you haven’t, but you have likely seen some images that use it. The plugin lets users play with edges and faces of an object parametrically, creating some fantastic shapes in the process.works with Cinema 4D versions R16-R19
